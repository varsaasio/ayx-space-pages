# ayx-space-pages

## 项目简介

`ayx-space-pages` 是一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态 HTML 文件的存储与托管空间，方便快速访问和分享已归档的页面内容。

## 目录说明

```
.
├── index.html          # 仓库主页，展示页面列表
├── pages/              # 存放归档的 HTML 页面文件
│   ├── example1.html
│   └── example2.html
└── assets/             # 资源文件（CSS、JS、图片等）
    └── style.css
```

- **pages/**：核心目录，所有独立 HTML 页面均存放于此。
- **assets/**：页面所需的静态资源，按需组织。
- **index.html**：可选入口页面，便于浏览和导航。

## 页面归档说明

- 每个 HTML 页面均为独立完整的文件，可脱离本仓库单独运行。
- 页面命名应简洁明确，避免特殊字符和空格。
- 归档内容不涉及任何具体域名、网站或商业推广信息。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/ayx-space-pages.git
   ```
2. 在 `pages/` 目录下添加新的 HTML 文件。
3. 更新 `index.html`（如有）以反映新增页面。
4. 提交并推送至 GitHub，即可通过 GitHub Pages 或其他静态托管服务访问。
